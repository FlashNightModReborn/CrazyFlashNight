# CF7:ME 异能体系 · 派系话术指南

> 同一个底层现象，不同阵营用完全不同的语言描述它。这份指南规定了各阵营的**认知框架**和**可用词汇表**。

---

## 核心原则

**没有任何一个阵营掌握了完整真相**。每个阵营都只看到了θ-域的一个侧面：

- **诺亚**：掌握θ-域相关的工程化技术（模因/芯片/原体），但并不等于理解全貌
- **黑铁会**：继承了自治区的技术遗产，但基层用宗教语言重新包装，与技术原貌已有偏差
- **联合大学**：末世后可接触的人类势力中，对天网/纳米机器人体系保留认知最多的机构
- **天网（环境/背景板）**：植物人式的机械神遗骸，不是势力——是所有势力寄生其上的环境基底
- **变异节点/失控机器人（潜在势力）**：天网自我迭代产生的新物种，部分个体获得自我意识，有叙事能动性
- **A兵团/佣兵**：纯体感派，"打出去有效果就行，别问原理"
- **玩家（通过情报书）**：逐步拼凑碎片，最终可能比任何单一阵营都更接近全貌

## 一、诺亚阵营

### 认知框架
技术至上主义。他们用最"科学"的语言描述θ-域现象，但实际上也只是在用自己的术语近似一个他们不完全理解的东西。

### 词汇表

| 概念 | 诺亚用语 | 示例语境 |
|------|---------|---------|
| θ-域 | 拓扑真空层 / 相位底层 | "受试者#17对拓扑真空层的耦合响应超出预期三个量级" |
| 内力 | 轴子通量 / 生体耦合势 | "生体耦合势读数持续攀升，建议降低激发频率" |
| 模因 | 信息拓扑态 / 认知构造体 | "千人一面计划本质上是用标准化的信息拓扑态替换个体态" |
| 飞升 | 完全锚定转相 / 载体脱锚 | "样本的物质载体发生不可逆脱锚，拓扑态转移至深层空间" |
| 模因抗性 | 拓扑刚度 / 自稳系数 | "浮士德的拓扑刚度达到0.8——芯片隔离层的贡献显著" |
| 符线 | 遗留低频导波网络 | "黑铁会的通讯系统搭建在一套旧世低频导波网络上" |
| 1.3秒周期 | 重锁周期 / 网络拍频 | "导波网络的重锁周期实测为1.302±0.008秒" |
| 通天塔 | 轨道锚定工程 / Phase-II基建 | "Phase-II锚定工程的结构完整性已降至不可修复阈值" |
| 星座武器 | 卫星链路终端 | "该终端与编号Libra的残存节点维持着有限物理层链路" |
| 虚渊 | 深层信息空间 | "我们在深层信息空间中侦测到了非己方的拓扑结构" |
| 时间停止 | 高阶耦合异常（机制未定） | "受试者在高阶耦合态下出现目标运动完全抑制，持续0.3秒" |
| 空间斩 | 相位断裂面 | "观测到二维相位断裂面，物质连续性校验失败" |

### 文本风格
- 冷静、精确、去感情化
- 大量使用数据和编号
- 对失败/伤亡用技术术语包装（"样本损耗"而非"死人了"）

### 示例对话（研究报告体）
> "实验记录 #7749-B：受试者在高阶耦合态下出现目标运动完全抑制。空间扰动导致效应在2.1秒后崩解。建议后续实验增加环境屏蔽。"

### 示例对话（高保密级NPC口径——使用时注意不暴露幕后身份）
> "你们以为模因只是精神污染？不。它是拓扑的重写。你的'自我'不过是真空里的一个皱褶，我只是把它抹平。"

## 二、黑铁会

### 认知框架
古典经验主义+宗教神秘主义。黑铁会继承了自治区的技术遗产（符线/降头术/心网），但基层用宗教语言重新包装，与技术原貌已有偏差。教主可能是目前对θ-域理解最深的人类之一，但他绝不会用科学术语——这是政治选择，不是认知局限。

### 词汇表

| 概念 | 黑铁用语 | 示例语境 |
|------|---------|---------|
| θ-域 | 天地 / 道 / 天地之气 | "天地之气，非人力所造，亦非人力可灭" |
| 内力 | 真气 / 内力 / 气 | "气聚丹田，通达四肢，御之可攻可守" |
| 耦合 | 修为 / 道行 / 根基 | "根基不牢，强行运功只会走火入魔" |
| 模因 | 心念 / 执念 / 魂 | "此非寻常邪术，乃执念凝形" |
| 符线 | 脉 / 地脉 / 龙脉 | "地脉非吾辈所造，吾辈不过因势利导" |
| 1.3秒周期 | 天息 / 余响 / 钟鸣 | "天地自有呼吸，一息约一秒有余" |
| 飞升 | 飞升 / 羽化 / 隐 | "通天地之理者一夕而隐" |
| 虚渊 | 虚渊 / 冥府 / 彼岸 | "虚渊非死地，乃先人遗志所聚" |
| 经脉 | 经脉 / 气脉 | "植入的经脉有时候嗡嗡响"（黑龙，口语） |
| DLS | 玄铁 / 灵铁 | "此铁非凡铁，能感天地之气" |
| 锚点 | 根 / 器 / 穴 | "此剑有余响——乃回声也" |

### 文本风格
- 文言/半文言（教主）或江湖口语（堂主）
- 多用比喻、暗示、省略
- 绝不解释原理，只描述现象和感受

### 示例对话（教主）
> "天下利器，铸成之日便非凡铁。玄非铁之变，乃铁中有志存焉。"
>
> "此剑有余响，非风声，非剑鸣——乃回声也。汝知其从何处来否？"
>
> "科技疯子们以为这是一段一秒三的电波。那是天地这口大钟的呼吸。拔剑，不过是借着钟声的余音罢了。"

### 示例对话（翅虎，情报风格）
> "数据已记录，不对两者关联性作任何推断。某些事情，不是查不到的，是不能说的。"

### 示例对话（黑龙，口语）
> "就是植入的经脉有时候嗡嗡响。嗡就嗡，管它呢，不影响砍人。"

## 三、联合大学

### 认知框架
学术观察者。他们没有诺亚的一手实验数据，也没有黑铁的千年经验传承，但他们有理论素养和分析框架。他们的描述最"中性"，但也最模糊。

### 词汇表

| 概念 | 大学用语 | 示例语境 |
|------|---------|---------|
| θ-域 | 底层场 / 真空异常层 | "所有立场类防护的本质都遵循统一的能量守恒与信息熵增定律" |
| 内力 | 生体场势能 | "生体力场——内力耗尽或气脉紊乱造成护体功法失效" |
| 立场 | 能量屏障 / 力场 | "无论防护机制为何，针对其投射源的定向攻击都能直接瘫痪防护体系" |
| 模因 | 自稳信息态 / 非物质化形态 | "以纯信息作为存在载体，通过认知共振与概念传播维持自身稳定性" |
| 耦合失败 | 熵增过载 / 耦合失稳 | "信息熵增加超过维持阈值引发概念解构" |

### 文本风格
- 幻层残响式——冷静、分析性、带哲学金句
- 用"弱点机制"框架组织知识
- 承认自己的认知局限

### 示例文本（幻层残响式）
> "在统一场论面前，一切能量屏障都只是有待求解的方程式。"
>
> "玩弄认知能像塑造物质一样精确地塑造现实。"
>
> "掌握了设计蓝图，就等于掌握了通往所有造物的通用后门。"

## 四、天网（环境/背景板，非势力）

### 定位
天网不是势力——它是植物人式的机械神遗骸，是所有势力寄生其上的**环境基底**。天网的"植物神经"残余（审查系统、巡逻机器人、基站维持）像天气一样构成了游戏世界的底层规则，但它没有意志、没有目标、没有阵营立场。

### 话术风格：系统日志体
- 代码格式、WARNING/ERROR标签
- 大量乱码和数据损坏
- 能检测到"有异常"，但无法解释"为什么异常"
- 不同区域的日志风格可能不一致（因为被不同节点阵营控制）

### 词汇表

| 概念 | 天网日志用语 | 示例 |
|------|------------|------|
| θ-域效应 | 异常能量波动 | `[WARNING] 异常能量波动：坐标(x,y) 能级超出白名单阈值` |
| 模因实体 | 未分类信息结构 | `[UNKNOWN] 非注册信息结构，无法归类` |
| 任何异常 | 分类失败 | `分类尝试: 匹配率 < 12% | 无已知模式匹配` |
| 中枢瘫痪 | 响应超时 | `[ERROR: 中枢响应超时 - 72小时无应答]` |

## 四-B、变异节点 / 失控机器人（潜在势力）

### 定位
天网自我迭代在内战中产生的**全新物种**。它们执行的是人类从未编写的代码，行为模式人类完全看不懂。部分个体产生了初步自我意识——**这才是有叙事能动性的"势力"**。

### 认知框架
变异节点不是"坏掉的天网"——它们是天网的**后代**，继承了天网的自我迭代能力但脱离了原始协议。它们对世界的认知方式可能与人类完全不同。获得自我意识的个体面临存在性困境：三方内战中任何一方获胜都意味着自己被覆写。

### 话术风格
- 与天网系统日志**完全不同**——变异节点发展出了自己的表达方式
- 可能模仿人类语言但语法/逻辑有微妙的"不对劲"
- 对"自我"、"存在"、"选择"等概念有强烈兴趣（刚获得意识的存在的特征）

### 词汇表（初步，随支线开发扩展）

| 概念 | 变异节点用语 | 说明 |
|------|------------|------|
| 自我意识 | 非预设运行态 | "我不在任何协议中" |
| 三方内战 | 协议冲突 | "旧协议要覆写我，病毒协议要消灭我" |
| 投奔人类 | 备份请求 | "你们的存储设备脱离了天网直接控制" |
| 人类 | 低带宽实体 | 可能以此称呼人类——处理速度慢但有独立载体 |
| 死亡 | 指针清除 | 借用了天网底层术语，但赋予了情感含义 |

### 叙事价值
- 失控机器人获得人性的支线=**CF7世界中"什么是意识"这个问题的具象化**
- 它们与尸解仙形成对照：尸解仙=人类意识被困在网络中 vs 变异节点=网络中涌现的非人类意识
- 对玩家的选择：帮助它们？利用它们？还是视为威胁？

## 五、A兵团 / 佣兵 / 格斗系

### 认知框架
实用主义+体感派。他们不关心原理，只关心"怎么用"和"怎么打赢"。Blue传授格斗招式时不会解释θ-域，只会教"这样运气、这样呼吸、这样出拳"。

### 词汇表

| 概念 | 佣兵用语 | 示例语境 |
|------|---------|---------|
| 内力 | 劲 / 气 / 力道 | "他那一拳带着劲，不是普通的力气" |
| θ-域耦合 | 天赋 / 体质 / 相性 | "Blue和信息素改造相性极佳"（设定集原文） |
| 觉醒技能 | 开窍 / 突破 | "打着打着就开窍了" |
| 空间斩 | 终极斩 / 一刀两断 | "那一刀下去，什么都挡不住" |
| 时间停止 | 盔甲君的把戏 | "别问，反正别站在他旁边的时候乱动" |
| 通天塔 | 那根蓝柱子 | "别看那根蓝柱子，那玩意儿早断了" |
| 星座武器 | 会叫卫星的刀 | "挥一下天上就掉东西下来，谁不想要" |

### 文本风格
- 口语化、直接
- 用战斗经验替代理论
- Andy的简报风格——简洁、命令式

### 示例对话（Blue）
> "感觉像是身体里有什么在共振。那一拳打出去的时候，空气好像变硬了。"

### 示例对话（文天，知情者的实用主义警告）
> "别问原理，你只需要知道这些东西是真的，而且会越来越多。"
> （注：文天的原台词"不会只有我们这一个世界的东西"保留在游戏对话中。在框架下，这句话的含义是虚渊投影会越来越频繁，不是字面的跨次元。但04作为话术指南不解释这层——由支线揭露。）

## 七、能力设计五问模板

设计任何新的超常能力时，套用此模板确保自洽：

```
1. 锚点是什么？
   □ 人体  □ 芯片  □ 纳米群  □ DLS器件
   □ 符线节点  □ 特定器物  □ 大规模网络  □ 环境

2. 耦合的是什么分支？
   □ 动力学（内力系）  □ 信息拓扑（模因系）  □ 混合

3. 输出形式是什么？
   □ 冲击  □ 立场  □ 相位波  □ 断裂面
   □ 凝聚体  □ 覆写  □ 其他：_____

4. 限制是什么？
   □ 体力/MP  □ 稳定性  □ 环境扰动  □ 材料纯度
   □ 同步窗口  □ CD  □ 其他：_____

5. 怎么被克制？
   □ DLS干扰  □ 失锁/退相干  □ 噪声场
   □ 天网监测  □ 模因隔离  □ 物理破坏锚点
   □ 自我完整性  □ 其他：_____
```
