# 武器散射度数值口径

本文档记录 `data/items/武器_*.xml` 中 `<diffusion>` 的设计口径，避免后续按单把武器直觉反复漂移。

## 字段含义

- `diffusion` 是子弹角度随机偏移的半径，实际弹道约在 `[-diffusion, +diffusion]` 整数角度内随机。
- 角色移动射击会在武器基础散射上追加惩罚，基础值不需要单独覆盖移动惩罚。
- `accuracy` 是命中加成，不等同于弹道散射。

## 建议带

| 类型 | 常规建议 | 说明 |
|------|----------|------|
| 狙击步枪 / 精确反器材 | `1-2` | 高射速、科幻半自动或粗糙旧枪可到 `3-4` |
| 战斗步枪 / DMR | `2-3` | 全威力自动射击平台可到 `4-5` |
| 突击步枪 | `3-6` | 短管、老旧、高后坐或粗放仿制可到 `7-8` |
| 冲锋枪 / PDW | `3-6` | MAC10、全自动手枪、粗糙短枪可到 `8+` |
| 手枪 | `3-5` | 竞赛枪、长管精密枪可到 `1-3`，土制/短管/粗糙枪可到 `7+` |
| 机枪 / 压制机枪 | `5-8` | 稳定架构或科幻补偿可到 `3-4` |
| 霰弹枪 | `8-20` | 锯短、超短枪可到 `25+`；独头弹、能量弹按实际弹种另判 |
| 发射器 | `2-6` | 火箭/导弹多为 `2-4`，榴弹多为 `4-6`，DIY 或弹幕武器可到 `8+` |

## 例外原则

- 可以因为现实原型、来源游戏手感、科幻设定、技能联动或平衡需求越出建议带，但需要在描述、注释或提交说明里留下理由。
- `GM6_LYNX` 的较大散射来自 CQB 反器材平台设定，允许保留。
- `NEGEV` 的极低散射是致敬 CS:GO 激光枪弹道，允许保留。
- `夜莺` 是能量霰弹武器，弹丸为能量束而非实体弹丸，不遵循常规霰弹枪扩散规律，允许保留极低散射。
- `T181` / `T181C` 为早期反器材原型枪，以粗犷结构和低精度换取大口径破坏力，允许保留偏大散射。
- `单管步枪` 是 1 级民用狩猎武器（老旧的单发猎枪），并非军用狙击平台，允许保留较大散射。
- 能量步枪、磁轨武器、G11 类超高速点射可使用极低散射，但应确认不是漏填默认值。
