# 彩蛋内容

部分原版内容虽然与重置版世界观难以兼容，但具有优秀或尚可的美术水平，这部分内容将划入彩蛋。

与此同时，重置组成员也在根据自己的喜好添加游戏内容，一部分是原创的新形象或用闪客系列作品素材拼接的形象，一部分是来自其他作品的形象。

**彩蛋内容不属于重置版世界观，但这并不影响它们作为游戏核心玩法的一部分，提供各具特色的游戏体验。**

大部分彩蛋内容放在迷之盔甲君的商店与技能学习栏，以及地图-偏僻寂静之处中。

## 划入彩蛋的原版内容

- 迷之盔甲君、爱国青年、迷路的忍者等NPC，以及对应的挑战副本

## 划入彩蛋的重置版内容

- 武装JK等部分重置版战宠
- 杰克霜精等NPC，以及对应的挑战副本
- 尾上世莉架等新的挑战副本
- 于迷之盔甲君处学习的超能力系列技能
- 由诺艾尔、heeho君等NPC售卖的装备
