# CF7:ME · 世界观权威表

> **文档性质**：治理层文档。只规定“某类事实以哪里为准”，不新增设定事实。
> **版本**：v0.1.0 · 2026-06-27（治理重构：把隐式 hub 与散文护栏改为显式路由）
> **使用方式**：写作或修改时，先查本表确定 canonical 定义点；非权威文档只保留必要指针，不重复维护完整定义。

---

## 一、权威表总则
> **稳定节名**：`20·权威表节`

1. **权威定义点唯一**：同一事实域只允许一个 canonical 定义点；其他文档可以引用、举例、剧情化表达，但不得写成第二个定义源。
2. **00 管层级，20 管路由**：00 负责“共享框架 / 当前主假说 / 支线真相”的归属分级与支线承载；20 负责“去哪里改这条事实”。
3. **稳定节名优先于行号**：跨文档引用使用 `NN·...` 稳定节名或文档名，不使用 `#L`、`:line`、裸行号等脆引用。
4. **行为保持**：本表只把既有事实的权威住所显式化，不把任何提案升格为硬 canon。

## 五、维护规则
> **稳定节名**：`20·维护规则节`

- 修改事实前，先在本表确认 canonical 定义点；若要改的是非权威文档，优先改成指针而不是扩写定义。
- 新增事实域时，必须同步补一行路由表，并在 00 标明层级。
- 删除或重命名稳定节名前，先跑治理校验；跨文档引用未迁移完不得合并。
- 发现两处都像权威定义时，处理顺序为：选唯一权威点 → 非权威处改为短指针 → 必要时补 validator 规则。
