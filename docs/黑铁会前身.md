自治区设定概要
时间线：二战后 → 三战前
C国西南，与叉叉重工（3xf）城乡并立，存在一个道教思想主导的巨型企业自治区，实行政教合一的封建式治理：

领导层（老祖宗们）：以"天人合一"为哲学框架，将道教修炼嫁接到赛博意识研究上，追求意识上传式的"尸解仙飞升"
核心层：指挥工厂与研究院，主攻神经连接与生化改造技术——这是后世降头术以及邪术改造的在黑铁会身上的技术源头
基层：大量公社-合同制农民，维持自治区的法理基础与经济运转——这是后世黑铁会的信众基础
自治区技术与武力极强，但采取闭关锁国策略：只要外界不打扰核心层修仙和做生意，就不对外扩张。对统合嘴上反对，实际无行动并且被事实上认作服从统合。

自治区与外部的关系
统合：自治区不构成现实威胁，乐见其因非战争原因自行解体
叉叉重工等邻居企业：有心无力，统合会阻止巨企之间的兼并战争，需要考虑国际观瞻
结果：自治区处于一种消极保护状态，没有外部势力趁虚而入
组织遗产
黑铁会复杂的政治设计骨架在自治区时代就已存在：

道教修炼位阶 → 宗法等级
企业部门制 → 会堂-职能挂钩
对不同下属单位的差异化管理 → 四种分坛制度
这保证了核心层消失后，中下层可以"遵循祖制"继续维持高组织力。

转折：Wings介入（三战初期）
Wings需要自治区的神经连接/意识研究技术，作为千人一面计划的基础。选择模因感染而非暴力手段——隐蔽性恰好不引起统合注意，看起来只像是内部宗教活动的极端化。

核心层被逐一忽悠，自以为实现了飞升，实际上意识被导入天网的某个隔离节点关了起来。

自治区解体后
核心层"飞升"消失 → 自治区管理真空
工农阶级事实自治 → 统合默许甚至庇护
农民层 → 多数派在神禧领导下，将自治区的组织遗产宗教化改写，创立黑铁教（少数派拒绝宗教化，保留集体劳动传统，后成为蓉都公社的农业底盘——详见07-八½）
工业层 → 保留完整技术传承，演化为铁枪会前身，延续自治区传统融入黑铁教
降头术/邪术 = 核心层遗留的神经连接技术与人体改造技术被不完全理解后的民间化产物
