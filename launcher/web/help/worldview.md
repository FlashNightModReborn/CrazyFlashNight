# 世界观

原版闪7前期剧情内容构建了一个优良但缺乏细节的设定框架。重置版剧情对于原版设定秉承取其精华，去其糟粕，致力于在原版前期剧情的基础上，构建丰富而自洽的世界观。为了让设定更为自洽，部分原版设定，尤其是后期剧情设定将被重置版剧情弃用。

## 重置版世界观将沿用的原版设定

- 闪客快打前六代的大部分设定
- 废城、堕落城、荒漠、禁区、诺亚前线基地、雪山等区域
- 在上述区域活动的组织，如A兵团、黑铁会、莱昂利亚革命军
- 狂野玫瑰、游寇、天网、诺亚方舟等
- 禁区森林，以及在该区域活动的异形与铁血战士

## 重置版世界观将弃用的原版设定

- 文明收割者相关设定
- 时间机器相关设定
- 虫洞、异界战场、坠毁战舰等区域
- 暴走、亚种、远古、史诗敌人等玄幻风格敌人换皮
- 远古诛神套装等玄幻风格装备换皮

> 上述部分内容将被划入彩蛋内容范畴，详见"彩蛋内容"页。
