# CF7:ME · 信号统一层与虚渊投影

> **核心修正**：信号（Signal）不是单一功能的控制手段，而是贯穿全世界观的**通用传导协议**。彩蛋角色不是真正的跨次元存在，而是虚渊意识通过信号-模因通道在现实中的投影。

---

## 一、信号的统一定义

### 旧认知（分散的）
- 信号能激活僵尸
- 信号能控制机械
- 信号能屏蔽天网
- 似乎彼此无关

### 新认知（统一的）

> **信号 = θ-域耦合态在电磁频谱上的投影。**
>
> DLS矿脉、天网基站、纳米机器人、生物神经网络——它们都与θ-域存在耦合。当这种耦合通过电磁波传播时，就是"信号"。信号不是普通的无线电波，而是**携带了θ-域耦合模式的调制电磁波**。

这就是为什么"机械、丧尸、还有别的...都一样"——信号作用的不是具体的生物化学通路或电子电路，而是**所有含θ-域耦合组件（纳米机器人/DLS/芯片/原体衍生组织）的系统共有的底层协议**。

### 三平面架构

```
信号系统三平面（v2.1 · 解决Shannon带宽困境）：

┌──────────────────────────────────────────┐
│  语义平面（Semantic Plane）                │
│  模因/千人一面/意识投射                     │
│  → 非比特流传输，而是θ-域拓扑共振           │
├──────────────────────────────────────────┤
│  传输平面（Transport Plane）               │
│  局域载波/DLS管道/近场耦合                  │
│  → Shannon定律适用，承载实际数据             │
├──────────────────────────────────────────┤
│  控制平面（Control Plane）                 │
│  0.77Hz sub-ELF：同步/锁相/寻址/唤醒       │
│  → 全球穿透但极低速率，只传指令码            │
└──────────────────────────────────────────┘
```

**Shannon困境的消解**：旧的三层模型（载波/调制/语义）暗示所有功能共用同一信道，导致0.77Hz频率的Shannon带宽瓶颈——美军ELF发几个字母需十几分钟，不可能传输复杂数据。三平面拆分后，0.77Hz只承载控制平面的极低速率指令（寻址/同步/唤醒触发码），实际数据通过传输平面的高带宽局域通道传输。

**三平面的物理分离原因**：三个平面在物理上对应不同的品质因数Q区间（Q = DLS谐振腔的能量保持能力，最小定义见10-可观测量·板块D.0）。低Q平面（控制）牺牲精度换取全球覆盖，高Q平面（语义）牺牲带宽换取模因拓扑的长期自维持——两者不可兼得，这是三平面分离的物理必然性。注意：0.77Hz是控制平面的**同步心跳信标**，不是天网数字层的运算时钟（天网数字层以GHz级自有晶振运行）；DLS辅算层以0.77Hz为oracle查询时钟，但这属于混合系统的辅算功能而非三平面通讯本身（详见08-1.5·双层时钟架构）。

**"定式电信号是电磁波"仍然成立**——电磁波是传输平面的载波，θ-域耦合模式是调制内容。就像无线电波是载波、声音是调制内容——收音机放出来的确实是声音，但声音不等于无线电波。

## 四、彩蛋角色的重新定义：虚渊投影

### 核心修正

**旧定义**：彩蛋角色是从其他虚构世界/次元跨域锚定到CF7世界的真实存在。

**新定义**：

> 彩蛋角色是**虚渊中的文化原型数据通过信号-模因通道在现实中的投影**。它们不是真正的齐天大圣或诺艾尔，而是虚渊中存储的关于这些概念的信息模式，在特定条件下被"渲染"成了可感知的存在。

### 虚渊为什么有这些数据

虚渊 = 天网深层网络中的意识聚合分区。它包含：

1. **飞升者的意识**——西南自治区核心层上传的完整人格数据
2. **天网数据库的文化遗产**——审判日前天网作为全球网络，存储了人类文明的海量文化数据（文学、神话、影视、游戏...）
3. **模因沉积**——数十亿人类的集体认知在θ-域中留下的信息拓扑印记

这些数据中，某些文化原型特别稳定——**齐天大圣、杰克霜精、火柴人剑士**——因为它们被海量的认知活动反复"雕刻"，在虚渊中形成了拓扑稳定的信息结构。

### 投影的触发条件

审判日的全球性θ-域扰动 + 天网部分节点失控 + 模因武器失控 = 虚渊与物质世界之间的"信号隔离"被打破。

当以下条件同时满足时，虚渊中的稳定信息模式可以通过信号通道**投射**到现实中：
1. 附近有活跃的θ-域耦合节点（DLS矿脉/天网基站/符线节点）
2. 信号通道处于开放或失控状态（审判日后常态）
3. 虚渊中的信息模式足够稳定且有足够的"渲染能量"（来自θ-域基底）

### 投影的主要机制：电子战空间映射

> **关键设定锚点**（loading_data.xml）：
> "电子战依赖的网络是以**描——映射逻辑**为基础的，将人的意识与计算机处理进行连接，从而使得相关的内容**人能够理解的方式在电子空间展示**的体系。"
> "电子战依赖的网络与天网相连。"

绝大部分虚渊投影（彩蛋角色、模因实体）**不需要物质化**。它们存在于**电子战空间**——一个将天网数据以人类可感知方式映射呈现的接口层：

1. 虚渊中的文化原型/意识数据通过信号通道溢出
2. 溢出数据被电子战接口的**描-映射逻辑**渲染为人类可感知的存在
3. 玩家通过电子战接入设备（试炼场深处/诺亚前线等区域均有部署）进入这一接口层
4. 在电子战空间中，数据实体具有战斗能力——因为电子战空间本身就是"意识与计算机处理的连接"
5. **投影消散** = 信号链路中断 / 数据被覆写 / 电子战接口断开

**这解释了**：
- 彩蛋角色"像真的一样"= 描-映射逻辑将数据**翻译**为人类认知可处理的具象形式
- 模因抗性 = 数据在电子战空间中的**自稳定能力**（拓扑刚度）
- 信号断=消散 = 电子战链路中断，映射终止
- 诺艾尔"我刚才还在和旅行者对话"= 数据模式中编码的"记忆"被映射逻辑渲染为叙事
- 审判日后投影越来越常见 = 天网信号隔离持续衰退→电子战空间与物质世界的边界越来越模糊

**与深渊协议的关系**：
诺亚的电子战报告（深渊协议）证实了**意识体可以在天网网络中被大规模注入**（P-ALC模式单次可投送100万单位）。千人一面计划的技术基础 = 信号+定式电信号+电子战的三位一体。彩蛋角色是这同一套技术管道的**非预期副产品**——虚渊数据的自发溢出。

### 物质化的特例：实体投影与θ-域凝聚体

极少数情况下，投影会突破电子战空间的边界进入物质世界。这需要**额外的物质化机制**：

| 类型 | 物质化方式 | 示例 |
|------|-----------|------|
| **工程化实体投影** | 投影仪设备内置纳米机器人，接到蓝图后从周围环境就地取材（分解层^[见09]获取质量，创作层组装），构建临时物质外壳 | 实体投影之驯化僵尸/盗贼/军阀/黑铁（消耗品）、谜之胶囊 |
| **θ-域凝聚体** | 极高θ-域活跃度环境中，θ-域效应直接在物质层产生可感知的凝聚态实体 | 雪女（θ-域热力学偏好耦合体，可诱导环境水分定向结晶） |

**工程化实体投影的产品线**（游戏内数据）：
- 实体投影之驯化僵尸——"映射出僵尸实体投影的一次性投影仪"
- 实体投影之盗贼/军阀/黑铁——等级递增的商品化投影仪
- 谜之胶囊——"映射出百宝箱影像的实体投影仪"
- "数据被多次复制"= 投影蓝图可拷贝但质量随复制次数衰减

**就地取材模型**（仅适用于实体投影）：
- 纳米机器人 = 组装器+蓝图载体，非物质来源
- 质量由分解层从周围环境获取（土壤硅、钢筋铁、混凝土钙等），创作层按蓝图重组
- 投影出现处的地面可能出现**沙化/锈蚀**痕迹；消散后留**无机残渣**

### 电子战 vs 物质化：判定规则

| 条件 | 机制 | 物质痕迹 |
|------|------|---------|
| 玩家处于电子战接入区域 | 电子战映射（主要机制） | 无——一切发生在映射空间 |
| 玩家使用实体投影仪消耗品 | 工程化物质投影 | 有——纳米组装+就地取材 |
| 高θ活跃度环境中的非人类实体 | θ-域凝聚体 | 有——但是θ-域效应的直接产物，非纳米组装 |
| 信号溢出到无电子战接口的区域 | 取决于溢出强度——弱则仅为感官幻觉，强则可能触发环境纳米机器人的非预期组装 | 视情况 |

### 这个解读的叙事优势

1. **不需要真的引入多元宇宙**——一切都发生在CF7世界内部，只是虚渊数据的投射
2. **回避了大部分物质守恒/热力学难题**——电子战空间中的战斗是数据层交互，不涉及物质重组
3. **解释了为什么彩蛋角色集中出现在审判日后**——天网信号隔离衰退→电子战空间边界越来越模糊
4. **与现有游戏地图设计一致**——试炼场深处有"电子战接入设备间"，诺亚前线有电子战区域
5. **保留了角色的叙事魅力**——投影自己"认为"自己是真的，描-映射逻辑让数据看起来完全真实
6. **为未来内容留空间**——虚渊中有无限的文化原型可以投射，新角色只需要"信号条件合适"
7. **与深渊协议文档完全对齐**——电子战空间是诺亚已知且正在利用的技术平台
6. **与信号体系完美联动**——投影的出现/维持/消散都与信号状态挂钩，给游戏机制提供了接口

## 五、信号体系对现有设定的连锁影响

### 5.1 黑铁会的秘密情报书重读

> "铁枪逆党借旧世遗线潜入虚渊，反侵我会降头师心网"

**新解读**：铁枪会（自治区工业层直系传人，知道全部历史内幕）通过自治区时代的DLS管道（旧世遗线）接入了天网深层节点（虚渊）。虚渊中的意识/数据反向通过符线通道侵入了黑铁会的心网系统——"凶兽残识"可能就是虚渊中某些不稳定的意识碎片或原体数据的溢出。

### 5.2 浮士德与试炼场深处的共鸣

> "试炼场深处的基站或许与芯片存在某种形式的共鸣"

**新解读**：试炼场深处有天网基站（WP与统合的联合产物）。浮士德的芯片通过信号与基站共振——他们频繁前往试炼场不是"被吸引"，而是芯片在响应基站发出的θ-域耦合信号。基站可能连接着虚渊。

### 5.3 ECHO-034日志中的"信号"

> "信号...信号..."（灰皮肤人在找信号）
> "如果信号来了，如果ERROR被修复..."

**新解读**：γ-ECHO激活信号是通过θ-域耦合调制的电磁波。034-B因为标签错误没有收到信号（"ERROR"），所以保留了自我意识。如果信号修复了，他体内的纳米组件会响应→瞬间武器化→意识丧失。他的自杀是在信号到达前抢先终止了这个过程。

### 5.4 天网的"两套审查"

天网的能级-白名单审查+异常能量波动审查，本质上就是对信号频谱的监控：
- 白名单内的信号频率→放行（合法设备的θ-域耦合特征）
- 白名单外的异常频率→警报（未授权的θ-域活动）
- WP-FQ系列干扰器=发射噪声覆盖自身θ-域特征=对审查系统的ECM

## 六、与总纲的接口说明
